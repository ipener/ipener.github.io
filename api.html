Furoa/API

The decentralized computing platform for AI training, CGI, HPE and Ethereum staking—at half the cost of existing services

south

AI, CGI, HPC, PoS—you name it.

Furoa is optimized for training neural nets like GPT-3, performing intense simulations like Finite Element Analysis, rendering complex CGI (Hi Tim), processing vast amounts of audio/video data, or running ASIC-resistant cryptocurrency mining and staking software.

The highest specced compute modules

are equipped with an 8-core ARM® v8.2 64-bit CPU, a 1024-core GPU, 32 dedicated Tensor cores and 16 GB of LPDDR5 RAM for a whopping 100 TOPS5—so yes, it can run Crysis. 10 Gb/s internet enables super fast data transfers. The compute modules and tiles are linked with PCIe Gen4 connectors and fibre-optic cables.

We’ll open-
source it

once the software is battle-tested because we deeply believe that a global computing platform should be built by the open source community, not corporations.

Get notified when Furoa API goes into private beta.

We expect to launch a developer beta program towards the end of 2022. The public release depends on the hardware production ramp which we expect to start in 2024. We aim to offer Furoa API at around half the cost of existing comparable compute services and aspire to make it even cheaper in the future.
